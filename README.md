## Descripción

**SEDADA** es un proyecto desarrollado en el SENA orientado a la creación de una plataforma web para la administración y optimización de servicios en restaurantes.

El sistema busca mejorar la gestión operativa mediante herramientas que permitan organizar procesos internos, controlar pedidos, administrar usuarios y optimizar la atención al cliente desde un entorno moderno, seguro y eficiente.

---

## Objetivos del proyecto

- Optimizar la gestión de servicios en restaurantes.
- Mejorar la administración de pedidos y clientes.
- Centralizar la información operativa en una sola plataforma.
- Facilitar el control y seguimiento de procesos internos.
- Ofrecer una experiencia eficiente e intuitiva para los usuarios.

---

## Funcionalidades principales

- Gestión de usuarios y roles.
- Administración de pedidos.
- Control de servicios.
- Gestión de clientes.
- Manejo de información en base de datos.
- Interfaz moderna y adaptable.
- Organización de procesos internos del restaurante.
